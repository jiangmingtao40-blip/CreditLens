# 征信报告AI解析服务 - 前端设计需求文档

## 一、项目概述

### 1.1 项目背景
基于 CRMEB 开源商城 + FastAPI + PaddleOCR + DeepSeek 的征信报告智能解析系统，为用户提供专业的征信报告上传、识别和分析服务。

### 1.2 核心价值
- **高效识别**：PaddleOCR精准提取报告文字
- **智能分析**：DeepSeek AI深度解读报告内容
- **风险评估**：提供专业的信用评分和风险预警

### 1.3 目标用户
- **个人用户**：需要了解自身信用状况的普通消费者
- **金融从业者**：需要快速分析客户信用报告的专业人士

## 二、功能需求

### 2.1 核心功能模块

| 模块 | 功能描述 | 优先级 |
|------|----------|--------|
| **首页** | 展示额度信息、核心功能入口、使用流程 | 高 |
| **报告上传** | 支持拍照/相册选择，支持JPG/PNG/PDF格式 | 高 |
| **OCR识别** | 自动识别报告内容，显示识别进度 | 高 |
| **AI分析** | DeepSeek分析报告，生成结构化数据 | 高 |
| **结果展示** | 展示信用评分、风险等级、详细分析 | 高 |
| **历史记录** | 查看过往分析记录 | 中 |
| **个人中心** | 用户信息、设置、客服入口 | 中 |
| **分享功能** | 分享给好友，支持分销佣金 | 低 |

### 2.2 页面结构

```
├── 首页（征信查询）
│   ├── 顶部品牌区
│   ├── 额度信息卡片
│   ├── 上传按钮（核心CTA）
│   ├── 核心功能展示
│   ├── 使用流程说明
│   └── 安全保障提示
├── 上传页
│   ├── 上传区域（拍照/选择图片）
│   ├── 文件格式提示
│   └── 上传进度
├── 分析页（加载中）
│   ├── OCR识别进度
│   └── AI分析进度
├── 结果页
│   ├── 信用评分展示
│   ├── 风险等级标识
│   ├── 详细分析报告
│   └── 风险预警提示
└── 个人中心
    ├── 用户信息
    ├── 历史记录入口
    ├── 客服入口
    └── 设置入口
```

## 四、页面设计详情

### 4.1 首页设计

**顶部区域**
- 渐变背景（主色调）
- 装饰性圆形元素（半透明）
- Logo + 品牌名称 + 标语

**额度卡片**
- 免费查询次数（大号数字）
- VIP标识（角标）
- 单次价格

**核心操作区**
- 醒目的"上传报告"按钮（主色渐变）
- 文件格式提示

**功能展示**
- 4个功能卡片（OCR识别、AI分析、信用评分、风险提示）
- 图标 + 名称 + 简短描述

**使用流程**
- 4步流程说明（上传→识别→分析→报告）

**安全保障**
- 安全认证标识
- 数据加密说明

### 4.2 上传页设计

**上传区域**
- 大尺寸上传按钮
- 支持拍照/选择图片
- 拖拽上传区域（可选）

**文件格式提示**
- 支持格式：JPG、PNG、PDF
- 大小限制：20MB

**上传状态**
- 进度条
- 上传中动画

### 4.3 结果页设计

**评分展示**
- 大字号评分数字（0-1000）
- 评分等级标签（优秀/良好/一般/较差）
- 雷达图/仪表盘可视化（可选）

**风险等级**
- 图标标识（安全/警告/危险）
- 颜色区分（绿/黄/红）

**详细分析**
- 个人信息提取
- 信贷记录汇总
- 逾期记录检测
- 风险预警列表

## 五、交互设计

### 5.1 微动画

| 场景 | 动画效果 |
|------|----------|
| **页面切换** | 淡入淡出、滑动过渡 |
| **按钮点击** | 缩放反馈（0.98倍） |
| **卡片悬停** | 阴影加深、轻微上浮 |
| **加载状态** | 骨架屏、脉冲动画 |
| **成功提示** | 弹出动画 + 图标跳动 |

### 5.2 加载状态

- **页面加载**：骨架屏占位
- **API请求**：loading遮罩或按钮loading状态
- **图片加载**：占位符 + 渐进式加载

### 5.3 错误处理

- **网络错误**：友好提示 + 重试按钮
- **文件格式错误**：红色提示框
- **服务异常**：提示用户稍后重试

- 敏感信息脱敏展示
